## Summary - 2026-06-06

- A large collection of miscellaneous experiments and mini-projects spanning Python, C++, Angular, Flask, OpenCV, TfL API, Raspberry Pi, and more.
- Topics include tube/bus data tools, image processing, games (poker, darts, GuessWho), web scraping, and various language/library explorations.
- Acts as a personal scratch-pad repo — individual folders are self-contained experiments rather than a unified application.

### angular/
- **bond-battler** — Angular app with a canvas-based game component.

### blog/
- C++ build options notes and a script (`stdafxize.py`) to automate adding precompiled header includes.

### boost/
- C++ Boost serialization experiments.

### cmake/
- CMake build system examples covering DLL hierarchies and lib/test/program structures.

### cpp/
- **binres** — Binary resource handling.
- **function_of** — Functional programming experiments in C++.
- **microtar** — Minimal tar archive implementation.
- **node** — C++ Node.js native addon experiments.
- **swig** — SWIG bindings between C++ and other languages.
- **tree_in_raw_storage** — Tree data structure in raw memory storage.
- **unlzh** — LZH decompression implementation.
- **variant** — C++ variant/union type experiments.

### maps/
- HTML5 canvas tile-based map viewer with a Python tile server and a JavaScript tile downloader (`jTileDownloader`).

### other/
- **basickl** — Basic KL language experiments.
- **ffmpeg** — FFmpeg usage experiments.

### processingjs/
- **darts_scoreboard** — Darts scoring visualisation built with Processing.js.

### python/
- **bus / tfl_api_debug** — Tools for querying the TfL API for bus and tube data.
- **tube_challenge** — Route planning and optimisation for the London tube challenge (visiting all stations), with GPS, OSM and SVG support.
- **poker** — Poker hand simulation and prime-number-based rank evaluation.
- **darts / livedarts** — Darts scoring logic and live scoring UI.
- **opencv / imagemagick / magickwand** — Image processing experiments.
- **GuessWho / games / game** — Guess Who solver and other simple games.
- **countdown_solver** — Solver for the Countdown numbers game.
- **flask** — Small Flask web app experiments.
- **findDups** — Duplicate file finder.
- **colorblend / PointToPixelLine / linear-predictor** — Graphics and maths utilities.
- **cdrom_autorip / poplib / ipc / coming_home** — Miscellaneous utilities (CD ripping, email via POP3, IPC, home automation).

### remote/
- Web-based remote keyboard interface — HTML/CSS keyboard UI with a Python POST handler.

### rpi/
- Raspberry Pi setup notes and utility scripts.

### svg/
- SVG pattern table generator.

### webcam/
- **time_lapse** — Webcam time-lapse capture utility.

### wikifilter/
- Filters Simple Wikipedia content by URL list (`simplewiki-filter.py`).

### worker_threads/
- HTML5 Web Worker examples including a prime number catcher demo.

### zim/
- Tools for creating and working with Zim offline wiki files, including an Open English Bible conversion.
